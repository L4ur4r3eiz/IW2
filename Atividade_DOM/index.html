1. O que é DOM?

DOM é a forma como o navegador organiza o HTML da página para conseguir entender e mexer nele. Quando a gente cria um site usando HTML, o navegador transforma tudo em uma espécie de estrutura em árvore, onde cada elemento vira uma parte que pode ser acessada Ele permite que o Js consiga encontrar esses elemntos e mudar coisas como texto, cor até adicionar novas coisas na pagina

2.O que é um nó (node)?

Um nó (node) é cada parte que existe dentro do DOM, ou seja, tudo na página vira um nó. Os elementos são as tags do HTML, como p, div e h1. Os atributos são informações extras dentro das tags, como id, class ou src. Já os textos são o conteúdo que fica dentro dos elementos, como um texto dentro de um parágrafo.

3. Explique os métodos: getElementById(), querySelector(), querySelectorAll()

O getElementById()

é usado para pegar um elemento pelo id, ou seja, ele busca aquele elemento único que tem um identificador específico. O

querySelector()

serve para selecionar o primeiro elemento que combina com um seletor CSS, como uma classe, id ou tag. O

querySelectorAll()

seleciona todos os elementos que combinam com o seletor, retornando uma lista com vários elementos.

4. Diferença entre innerHTML e innerText

innerHTML

pega ou altera o conteúdo de um elemento incluindo as tags HTML, ou seja, ele interpreta o código. Já o

innerText

pega ou altera apenas o texto visível, sem interpretar as tags, mostrando tudo como texto normal.

5. O que são eventos?

Eventos são ações que acontecem na página, geralmente feitas pelo usuário, e que o JavaScript consegue identificar e responder. O onclick acontece quando o usuário clica em algum elemento. O onchange acontece quando um valor é alterado, como em um input ou select. O onmouseover acontece quando o mouse passa por cima de um elemento.
